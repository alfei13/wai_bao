## 二、分步运行命令

### 1. 编译 APK
```bash
cd /Users/zhengfeifei/Desktop/waibao3
./gradlew assembleDebug
```

### 2. 安装到手机
```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### 3. 重启无障碍服务（每次安装后必须执行！）
```bash
adb shell am force-stop com.waibao.floatingcollector
sleep 1
adb shell settings put secure enabled_accessibility_services '""'
sleep 1
adb shell settings put secure enabled_accessibility_services com.waibao.floatingcollector/.CollectorAccessibilityService
adb shell settings put secure accessibility_enabled 1
sleep 1
```

### 4. 启动 App
```bash
adb shell am start -n com.waibao.floatingcollector/.MainActivity
```

### 5. 启动触屏辅助脚本（菜亿萝采集需要！）
```bash
cd /Users/zhengfeifei/Desktop/waibao3
bash tap_helper.sh &
```

## 三、触发自动采集命令

### 在主页点击按钮采集
打开 App → 点击「大润发优鲜」或「菜亿萝」按钮即可。

### 用命令行触发自动采集
```bash
# 大润发优鲜
adb shell am broadcast -a com.waibao.floatingcollector.AUTO_COLLECT --es app darunfa

# 菜亿萝
adb shell am broadcast -a com.waibao.floatingcollector.AUTO_COLLECT --es app caiyiluo
```

### 手动采集当前页面
点击悬浮球 → 展开面板 → 点「采集当前页面」

或命令行：
```bash
adb shell am broadcast -a com.waibao.floatingcollector.COLLECT
```

## 六、常见问题

### Q1: 无障碍服务没生效？
重新执行步骤 3 的命令。

### Q2: 菜亿萝搜索点了没反应？
确保 tap_helper.sh 正在运行（步骤 5）。它负责监听 logcat 中的 TAP_REQUEST 标记并执行真实的 adb tap。

### Q3: 悬浮窗不显示？
检查无障碍服务是否开启（步骤 3），或执行：
```bash
adb shell am force-stop com.waibao.floatingcollector
adb shell settings put secure enabled_accessibility_services '""'
adb shell settings put secure enabled_accessibility_services com.waibao.floatingcollector/.CollectorAccessibilityService
adb shell settings put secure accessibility_enabled 1
```

### Q4: Git 推送失败？
使用本地代理（ClashX 默认端口 7890）：
```bash
git -c http.proxy=http://127.0.0.1:7890 -c https.proxy=http://127.0.0.1:7890 push origin main
```

## 七、一键运行脚本（可选）

将以下内容保存为 `run.sh` 到项目根目录：

```bash
#!/bin/bash
set -e
cd "$(dirname "$0")"

echo "[1/5] 编译 APK..."
./gradlew assembleDebug

echo "[2/5] 安装 APK..."
adb install -r app/build/outputs/apk/debug/app-debug.apk

echo "[3/5] 重启无障碍服务..."
adb shell am force-stop com.waibao.floatingcollector || true
sleep 1
adb shell settings put secure enabled_accessibility_services '""'
sleep 1
adb shell settings put secure enabled_accessibility_services com.waibao.floatingcollector/.CollectorAccessibilityService
adb shell settings put secure accessibility_enabled 1
sleep 1

echo "[4/5] 启动 tap_helper（菜亿萝）..."
pgrep -f "tap_helper.sh" || bash tap_helper.sh &

echo "[5/5] 启动 App..."
adb shell am start -n com.waibao.floatingcollector/.MainActivity

echo "✅ 完成！"
```

赋予执行权限：
```bash
chmod +x run.sh
```
